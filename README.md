# Sobre
<p>Um pokedex simples que utiliza a PokeApi, uma api com dados de pokemons.</p>
<img src = "./public/pokedex.gif" alt = "pokedex.gif" /> 

# Dependência
```bash
npm install
```

# Rodar aplicação
<p>Após instalar a dependência</p>

```bash
npm start
```
